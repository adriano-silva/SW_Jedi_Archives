# SW Jedi Archives

Simple web app in Ruby on rails displaying some data from https://swapi.dev/ API

* Ruby version
 2.6.6

* Node version
13.7.0 

* SQLite3 version
3.8.7.2

* Rails version
5.1.7

* System dependencies
 - Ruby
 - Node.js
 - SQLite3

* Configuration
 Open terminal and run "Rails server"
 Then access page http://localhost:3000/login
